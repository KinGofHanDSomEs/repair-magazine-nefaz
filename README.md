# Журнал ремонта техники «НефАЗ»


## Информация

#### Данный проект нацелен на создание удобной организованной площадки для ремонта техники компании «НефАЗ»


## Запуск

#### Для запуска измените конфигурацию в файле .env.example и переименуйте в .env
#### После введите команды в разных терминалах:
```php artisan serve```  
```npm run dev```
